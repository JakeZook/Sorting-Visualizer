# Algorithm Visualizer

A source of information to better understand sorting algorithms.
- [Deployed Project](https://visualizerjz.netlify.app)

## Features

- Sorting
  - Bubble
  - Megre
  - Insertion
  - Quick
  - Selection
- Searching
  - Linear
  - Binary
  - Jump
  - Interpolation
  - Ternary
- Pathfinding
  - A*
